# 萌卡YGO竞技助手

一款安卓端萌卡™YGO竞技平台的助手程序

## 主要功能

- 观战列表&自定义对局推送

- 竞技数据显示&竞技匹配
- 前一百玩家排行榜

## 工具

- 手牌概率计算器
- 自定义房间助手
- 先行卡自动下载安装
- *（更多功能开发中...）*
